## About

The project for Rolling Scope School Stage 2 

Deploy: https://seala11.github.io/online-store/

## Stack

HTML, Sass, TypeScript, Webpack, EsLint, Jest

## Setup and Running

### Install dependencies:

```bash
$ npm install
```

### Running the app

```bash
$ npm run start
```

### Prepare for production:

```bash
$ npm run build
```

### Linter check:

```bash
$ npm run lint
```

### Jest tests:

```bash
$ npm run test
```
